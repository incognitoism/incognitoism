## Featured Work

### 🔬 SA-DSVN — Shower-Aware Dual-Stream Voxel Networks for Cosmic-Ray Muon Tomography

> First-author research paper introducing a dual-stream 3D convolutional architecture for structural defect detection in muon tomography volumes. **arXiv preprint live; conference paper under peer review.**

- Designed and implemented a **dual-stream 3D CNN with cross-attention fusion** for six-class voxel segmentation across muon-tomography volumes
- Achieved **96.3% voxel accuracy** and **100% volume-level detection sensitivity** on a held-out test set of 60 independently generated volumes
- Established **electromagnetic-shower multiplicity as a learned feature**, raising defect-mean Dice by **+28%** over scattering-only baselines
- Co-authors: Sambhav Agarwal, Palash Dutta (project supervisor), Raja Karmakar (HOD CSE), Sudeshna Goswami

**Stack:** `Python` · `PyTorch` · `Geant4` · `NumPy` · `Cross-attention 3D CNNs`

### 🌱 RootSense — End-to-End IoT and ML Platform for Precision Agriculture

> Bachelor thesis project: a four-tier platform combining edge sensing, ML-driven recommendations, and a mobile interface for smallholder Indian farms.

- **Five-student group project** (supervisor: Prof. Palash Dutta · HOD: Prof. Raja Karmakar) — Project-I completed Sem 7 (graded O / 4 credits); Project-II in progress, expected July 2026
- Designed and implemented the **analytics pipeline**: three tree-based ML models trained on a 20,000-sample dataset calibrated against ICAR / TNAU / KVK reference standards
  - **Random Forest soil-health regressor** — R² = 0.991
  - **XGBoost crop classifier** — 98.65% accuracy
  - **Random Forest soil-type classifier** — 97.42% accuracy
- Integrated **SHAP TreeExplainer** for per-prediction explainability across all three models
- Architecture: **LoRa-mesh sensor layer → Raspberry Pi edge gateway → AWS serverless backend (IoT Core, Lambda, Timestream, AppSync, SageMaker) → React Native frontend**

**Stack:** `Python` · `XGBoost` · `Scikit-learn` · `SHAP` · `Raspberry Pi` · `LoRa` · `AWS Serverless` · `React Native`
